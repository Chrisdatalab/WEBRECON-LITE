# WebRecon-Lite

A lightweight Python web reconnaissance tool for crawling websites, extracting links, and generating simple reconnaissance reports.

## Features

- Crawl web pages
- Extract links and URLs
- Generate recon reports
- Wordlist-based path discovery
- Modular project structure

## Technologies

- Python 3
- Requests
- BeautifulSoup
- Regular Expressions

## Example Usage

python main.py https://example.com

This project is intended for educational and authorized security testing purposes only.
